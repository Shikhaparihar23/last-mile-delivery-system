# Last Mile Delivery Management System

A MERN Stack based Last Mile Delivery Management System.

## Features

- User Authentication (JWT)
- Admin Dashboard
- Customer Dashboard
- Agent Dashboard
- Create Package
- Assign Package
- Track Package
- Delivery Status Updates
- Role-Based Access Control

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Socket.io
- JWT Authentication

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm start
```
